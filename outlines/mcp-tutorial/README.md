# MCP 学习路径（8 小时）

> 适用对象：有 AI Agent 基础（理解 tool-use、JSON-RPC 等概念），想深入理解 MCP 协议并在 Java/Spring 微服务中落地的开发者。
> 语言偏好：**中文视频 + 英文官方文档**。中文视频用于建立直觉，官方文档用于技术精读。
> 技术栈：基础实操用 TypeScript（MCP 官方 SDK 主推），最终章用 Java/Spring（Spring AI MCP）。

---

## 这份路径是什么 / 不是什么

**是**：
- 一份按「任务 → 资源 → 产出」粒度写好的执行手册
- 资源经过筛选：官方文档优先，中文视频只选有独立思考的（过滤掉 LLM 洗稿）
- 8 小时内能完成，每个任务标注预计时间

**不是**：
- 一份完整的 MCP 讲义——本路径假设你会自己读英文文档，它的价值在于**筛选 + 排序 + 动手题**
- 一份面向零基础的入门——你需要已经理解 AI Agent 的 tool-use 机制

## 资源质量标准

沿用 agent-learning 路径的原则：

1. **一手源优先**：MCP 官方文档 / Anthropic 博客 / Spring AI 官方文档 > 中文翻译 > 中文二手分析
2. **视频是辅助不是主食**：B 站中文视频用于「建立直觉」和「跟完一遍实操」，不替代阅读官方文档
3. **不做资料收集瘾**：每章只给必要资源，读完就做，不囤
4. **避雷**：标题含「保姆级 / 万字长文 / 全网最细 / 一文搞懂 / 2026版」的默认不读
