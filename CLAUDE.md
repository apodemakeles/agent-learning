# CLAUDE.md — AI Agent 研究与学习仓库

本仓库用于 **AI Agent 研究、学习与知识整理**，不是软件开发项目。默认以 **资料、大纲、问题与文档总结** 为主，**不要**主动引入或扩展实现代码，除非用户明确要求。

## 仓库定位

| 用途 | 说明 |
|------|------|
| 学习资料 | 论文笔记、课程链接、摘录、术语表 |
| 大纲 | 主题/章节结构、学习路径、待读清单 |
| 问题 | 开放问题、待验证假设、讨论提纲 |
| 文档总结 | 综述、对比表、一页纸摘要 |

## 与助手协作时的偏好

1. **优先文字与结构**：回答以解释、对比、清单、大纲为主；需要示例时用伪代码或概念性描述即可，避免默认新建可执行代码仓库。
2. **引用与可追溯**：整理或总结时，尽量注明来源（论文名、章节、链接、日期），便于日后核对。
3. **粒度与格式**：长内容用标题分层；对比类内容优先表格；复杂流程可辅以简短列表或 ASCII/mermaid（仅在用户需要时）。
4. **不主动扩张范围**：不擅自添加「示例项目」「脚手架」或与本仓库学习目标无关的代码目录。

## 若用户明确要求写代码

仅在用户 **明确** 提出（例如「写一个脚本处理这份笔记」「做个小 demo」）时，再添加或修改代码；并说明目的与范围，避免把本仓库变成默认代码库。

## 可选目录约定（可自行调整）

若后续需要整理文件，可沿用类似结构（非必须）：

- `notes/` — 笔记与摘录  
- `outlines/` — 大纲与学习路径  
- `questions/` — 问题清单  
- `summaries/` — 总结与综述
